# BuddyHub Color Menu

Choose a verified Buddy color:

- `/buddyhub:color-green`
- `/buddyhub:color-orange`
- `/buddyhub:color-blue`
- `/buddyhub:color-pink`
- `/buddyhub:color-purple`
- `/buddyhub:color-red`
- `/buddyhub:color-black`
- `/buddyhub:clear-color`

Then run:

- `/buddyhub:apply`

Use `/buddyhub:inspect` if you want to check the effective selection before applying.
